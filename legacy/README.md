# Prettier - Code formatter (Deprecated)

> **This extension is deprecated. Please use [esbenp.prettier-vscode](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) instead.**

## Official Extension

The official Prettier VS Code extension is:

**[esbenp.prettier-vscode](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)**

Install via the command line:

```bash
code --install-extension esbenp.prettier-vscode
```

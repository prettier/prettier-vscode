# Contribute

Feel free to open issues or PRs!

## Running extension

- Open this repository inside VSCode
- `yarn install`
- Debug sidebar
- `Run Extension`

## Running tests

Tests open a VSCode instance and load `./testProject` as root workspace.

- Open this repository inside VSCode
- Debug sidebar
- `Launch Tests`

OR

Without having an instance VSCode running (or it won't start)
`yarn test`
